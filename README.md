# Welcome to todo-app-rn

This is a simple todo app created using React Native + Expo. It allows you to create your todos, update them and delete.

# Features

1. Create Todos
2. Update status of created Todos
3. Delete Todos if needed
4. Switch theme between light, dark and system

# Upcoming features

1. Enhanced UI
2. Support edit Todos
3. Storing todos into Database
4. Real time sync

# Steps to install

```bash
# Clone respository
git clone https://github.com/swapnasahoo/todo-app-rn.git

# Move to project folder
cd todo-app-rn

# Install dependecies
npm install

# Run development server
npx expo start --clear
```
